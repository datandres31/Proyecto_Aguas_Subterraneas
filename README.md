# Proyecto_Aguas_Subterraneas
Este repositorio alberga todos los archivos de origen, notebooks y documento técnico del proyecto construido para identificar usuarios con patrones atípicos de consumo  y concesión de aguas subterráneas para la sabana de Bogotá.

Los archivos de origen de los datos incluye:
1.Reporte de Concesión de aguas subterraneas de la CAR Cundinamarca
2.Reporte historico de niveles freaticos de los pozos de monitoreo de aguas subterraneas de la CAR en la sabana de Bogotá.
